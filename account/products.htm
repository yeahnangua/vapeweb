Frequently asked questions

How do I run Vape?

1. Launch Minecraft

  • Open your Minecraft launcher, and launch Minecraft on a Minecraft version supported by Vape. You can view all supported versions on another section within this same FAQ.
  • For greatest compatibility, this should be the default Minecraft launcher while running on Forge.
  • The latest common Forge versions can be found here: 1.8.9 | 1.7.10 | 1.12.2 | all.
    To install Forge, just close your launcher, double clicker the Forge installer, and open the launcher again. There should now be a launcher profile created that you can launch.
  • Default "Vanilla" Minecraft is also supported for certain veresions, check the FAQ for supported vanilla versions.
    Any modded vanilla versions such as Optifine should also work fine.
  • You can attempt to run on 3rd party clients/launchers, but Vape is not guaranteed to run on it, as Vape does not support any 3rd party clients/launchers.

2. Double click the Vape exe to launch

  • Double click the Vape exe while Minecraft is open.
  • You may be prompted for a login, use the same credentials used to login to the Vape website.
  • After authentication the injection process will start. If you crash or get stuck while loading please submit a contact form and let us know so that we can help you resolve the issue.

3. Open the GUI

Vape V4:

  • Press the GUI keybind while in game to open the GUI. Make sure you are actually in game, not in a menu.
  • The default GUI keybind is Right Shift.

Vape Lite:

  • On Vape Lite the GUI is automatically opened after loading has completed.


Once you have the GUI open you can use all features of Vape.


Everything from here should be mostly self explainitory. But if you do have any questions do not hesitate to reach out to us for help, or check our forums for advice.

What is the difference between Vape V4 and Vape Lite?

Vape V4

Vape V4 includes a wide variety of features(50+). This includes all ghost features, along with several blatant hacking features. Vape V4 can be used anywhere but is perfect for PVP Minigames servers, as well as practice servers.

Vape V4 does not protect against screenshares.

Vape lite

Vape Lite only includes the essential ghost features for closet cheating. Vape Lite is mostly geared towards practice servers, but can easily be used anywhere PVP is involved.

Additonally, Vape Lite protects against screenshares.

What is the difference between "Undetectable" and "Screenshare Proof"?

Undetectable means that the features within the client are not inherently detectable by an anticheat, generally. However, you are still capable of increasing the settings to the point where an anticheat can easily detect them.

Screenshare Proof means that the client is designed to be difficult to find in a screenshare, and that under most circumstances you will bypass screenshares with it.
